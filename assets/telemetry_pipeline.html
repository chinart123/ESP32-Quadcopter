Project Windify: USART1 Telemetry Pipeline

Data flow from MPU6050 sensor to PC Ground Station (STM32F103 Single-Core Architecture)

1. INPUT SENSOR
  • MPU6050 (IMU)
  • 6-DOF Raw Data
  • Hardware Address: 0x68
➔
2. DATA ACQUISITION
  • STM32 I2C1 Bus
  • 400 kHz Fast Mode
  • Single-core blocking read
➔
3. FIRMWARE LOGIC
  • Sensor Fusion
  • Complementary Filter
  • Format to CSV: roll, pitch, yaw, pwm, i2c_bps
➔
4. TRANSMISSION
  • USART1 (PA9 TX)
  • 115200 Baud Rate
  • fputc retargeted
  • 5 Hz Packet Rate
➔
5. GROUND STATION
  • PC Terminal
  • Hercules Software
  • Live data display
  • Excel .csv log analysis
